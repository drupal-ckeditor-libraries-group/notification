Notifications Dev Sample

Play with notification using buttons. Note that if you close progress notification it will be shown again when upload is done.

You can also play with notifications using console: 

var editor = CKEDITOR.instances.editor1;

var warning = editor.showNotification( 'Foo', 'warning' );
warning.hide();

var progress = editor.showNotification( 'Bar', 'progress', 0.1 );
progress.update( { progress: 0.5 } );
progress.update( { type: 'success', message: 'Done!', important: true } );
